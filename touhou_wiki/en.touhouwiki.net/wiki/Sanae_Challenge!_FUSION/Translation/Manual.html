// 概要
//------------------------------------------------------------//
『サナエチャレンジ！FUSION』は、上海アリス幻樂団様制作の同人作品
「東方Project」を題材にしたアクションゲームです。

プレーヤは東風谷早苗（さなぽん）を操作し、ひたすらケガレを祓い
つつ分社を設置しまくって信仰をモリモリ増やしていくのが目的です。

まず最初にプレイ面を選択し、各LAP毎に出現する拠点を制圧していき
ます。全8LAPを通過すればクリア達成となり、選択できるプレイ面が
増えていきます。

◇布教開始 GAME START
　　このゲームのメインコンテンツです。
　　面選択の後、ゲームを開始します。

◇布教記録 RANKING
　　これまでのスコアをランキング表示します。

◇記録再生 REPLAY
　　過去に記録したプレイ内容を再生します。

◇設定変更 OPTION
　　ゲームの各種設定を変更します。（詳細は「オプション説明」参照）

◇宝物庫 SPECIAL
　　情報の閲覧や、特殊モード設定を変更します。

◇終了 EXIT
　　ゲームを終了します。

◇移動：          ←or→

◇アタック(A)：   ボタン1

◇ジャンプ(J)：   ボタン2

◇オーダー(O)：   ボタン3

◇テキスト(T)：   ボタン4

◇ポーズ(強制)：    F5
◇リセット(強制)：  F1



//------------------------------------------------------------//
// ステータス説明
//------------------------------------------------------------//
◇穢レーダー
　　中心の水色の点がプレーヤ、赤色の点が敵（ケガレ）、
　　黄色の点が拠点、緑色の点がアイテムを示しています。

◇信仰率
　　プレーヤのライフに該当します。
　　信仰率は常に減少し続けます。０％になるとゲームオーバーです。
　　また敵に接触した場合も大きく減少し、全体の何割かを失います。
　　信仰率を上昇させる手段は、ミソギを行って分社を建てるしかあり
　　ません。
　　信仰率が高い程、信仰点の上昇具合も跳ね上がります。

◇信仰点
　　スコアに該当します。
　　ケガレを気絶させる、カミカゼでぶつける、ミソギを成功させる等
　　のアクションで加算されていきます。。
　　またゲーム終了後のリザルトでも加算されます。

◇調教LV
　　お空と早苗の仲良し度合が表示されます。
　　得点加算には影響しませんが、お空の命令受付スパンが短縮されます。
　　命令を行ったり、光り物アイテムを取得するとLVが上がります。

◇最大信仰率
　　ゲーム中、一番高かった時の信仰率が表示されます。
　　リザルト時の得点加算に影響します。

◇平均信仰率
　　ゲーム開始から現在までの信仰率の平均値が表示されます。
　　リザルト時の得点加算に影響します。

◇LAP数
　　現在のLAPが表示されます。
　　1つの面につき、全8LAPで構成されています。

◇テキストウィンドウ
　　特定のタイミングで自動的に表示されます。
　　神奈子、諏訪子との会話や、購読中の新聞記事等が表示されます。

◇FPS
　　画面右下の数字です。
　　ゲームの進行速度が表示されています。
　　60付近をマークしていれば快適に動作している事になります。



//------------------------------------------------------------//
// アクション説明
//------------------------------------------------------------//
◇ジャンプ
　　ジャンプします。ボタンを押す長さで高さが変わります。
　　本作品では、自由に連続ジャンプを行う事ができます。


◇ハラエ　　　ボタン1（レバー任意方向）
　　放物線上にショットを撃ちます。
　　レバーによって、地上5、空中8方向に撃ち分ける事ができます。
　　ハラエ中も自由に移動ジャンプが行える他、ボタンを押し続ける
　　間は向きを固定したまま移動攻撃する事ができます。

　　気絶中の敵にハラエを撃ち続けると、浄化させてしまいます。


◇カミカゼ　　↓＋ボタン1
　　御祓いを始め、眼前に吸引属性の神風を発生させます。
　　ボタン押し続ける間持続し、神風に触れた敵を吸い寄せます。

　　敵を吸った状態でボタンを離すと、回ってた敵を全て一斉に射出し、
　　他の敵にぶつけて大ダメージを与える事ができます。
　　カミカゼを食らった敵はさらに弾丸となって吹っ飛びます。

　　この攻撃を受けた敵はハラエと異なり絶対に浄化されません。


◇ミソギ　　　↓＋ボタン1（拠点に接近した状態で）
　　ケガレ拠点の前でカミカゼを実行すると自動的にこれになります。
　　モーションは同じですが、カミカゼの様な吸引判定を持ちません。
　　その代わりに、これを行う事で拠点を浄化する事ができます。

　　ボタンを押し続ける間専用の浄化ゲージが出現し、徐々に増幅、
　　ゲージが満タンになれば浄化が成功し、奇跡の力で分社を建造します。
　　その瞬間、この拠点から出現していた敵全てが一斉に浄化されます。

　　成功と共に、信仰率が上昇します。上昇の度合は、拠点から出現
　　していたケガレの数によって変動します。
　　またミソギが成功した瞬間から暫くの間は無敵属性が付きます。

　　ちなみに途中で攻撃受けたりボタンを離したりすれば中断します。


◇命令選択　　ボタン3長押し（↑↓←→で選択、ボタン離しで発動）
　　お空に命令する内容を選択します。
　　お空側が準備OKになっていないと発動できません。

　　・「破」 …　核熱球で前方に攻撃を行わせます。
　　・「防」 …　周囲にバリアを展開させます。
　　・「飛」 …　お空に乗り、自由飛行を行います。
　　　　　　　　 方向キー＋ボタン2で突進攻撃を行います。
　　・「核」 …　フュージョンを実行させます。
　　　　　　　　 チレイが存在しない場合は発動できません。


◇GOD HAHURI発動　　ボタン1+ボタン2同時押し（信仰率100%以上必要）
　　GOD HAHURI発動を行うと、常識に囚われなくなり超パワーアップします。
　　ステージ中1回しか発動できないのでご注意ください。

　　・ハラエが超強化され、ケガレを浄化するだけで信仰率が上昇します。
　　・ミソギ、フュージョン命令は行えなくなります。
　　・一定時間で終了し、終了時に信仰率が100%マイナスされます。



//------------------------------------------------------------//
// システム説明
//------------------------------------------------------------//
◇ケガレと拠点
　　ケガレ（穢）とは不浄なる悪しき精神の具現であり、本作における敵です。

　　ケガレ単体は基本的にゆらゆら寄ってくる（触れるとミス）だけなので単体
　　では大した脅威にはなりませんが、その総数が非常に多く、兎に角沢山の群
　　れで襲ってきます。

　　マップの至る所に点在する、紫色のもやを「拠点」と呼びます。
　　ケガレはこの拠点から一定周期でひたすら沸いてきます。
　　ミソギで拠点を浄化しない限り、ケガレの増加は止められません。

　　裏を返せば、この拠点が早苗達に対する信仰を妨げている訳なので、
　　此れを浄化さえすればその分信仰が増える、という事にもなります。


◇チレイとフュージョン
　　チレイ（地霊）は今作よりケガレに紛れて登場する、新たな敵です。

　　チレイ単体はケガレ以上に非力ですが、ケガレとの最大の違いは、通常の浄
　　化が通用しません。ハラエでもミソギでも、消滅させる事ができません。

　　チレイを浄化するには、お空の命令選択でフュージョンを実行させます。
　　フュージョンでチレイを集めたお空にミソギを行う事により、チレイの浄化
　　が成立となり、信仰率と信仰点が上昇します。


◇テキストウィンドウ
　　特定のタイミングで自動的に表示され、様々な情報を表示していきます。
　　表示中は、Tボタンでページ送りを行えます。
　　OPTIONで表示・非表示の切替が可能です。

　　神様通信：　　神奈子、諏訪子と会話を行います。
　　　　　　　　　攻略のヒントなどが表示される事もあります。
　　文々。新聞：　新聞の記事をランダムでピックアップし表示します。



//------------------------------------------------------------//
// オプション説明
//------------------------------------------------------------//
◇エフェクト設定
　　「補正あり」の場合、エフェクトを滑らかに補正して描画します。
　　「補正なし」の場合は補正を一切行いません。

◇BGM音量
　　BGMの音量を調整します。

◇SE音量
　　効果音の音量を調整します。

◇ボタン設定
　　パッドコンフィグを行います。

◇テキストウィンドウ
　　テキストウィンドウの表示/非表示を設定します。
