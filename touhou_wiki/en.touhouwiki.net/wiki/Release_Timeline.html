This page lists all first-party Touhou Project-related releases, news, and events. Note that this is not a timeline of chronology of when events happen in canon; see Gensokyo Timeline for that.

There is a more up to date version of this page on the Japanese wiki.

2001 and Prior[edit]

Date Type Event
1996/11/--/ Exhibit Highly Responsive to Prayers first displayed (20th Hatoyama Festival)
1997/08/15 Game Highly Responsive to Prayers and Story of Eastern Wonderland released (Comiket 52)
1997/12/29 Game Phantasmagoria of Dim.Dream released (Comiket 53)
1998/08/14 Game Lotus Land Story released (Comiket 54)
1998/--/-- Touhou is featured in GameLab
1998/11/05 Exhibit Shuusou Gyoku Prototype ("Giant Project") first displayed (22th Hatoyama Festival)
1998/12/30 Game Mystic Square released (Comiket 55)
1999/03/-- ZUN graduates college, joins Taito Corporation
1999/11/05 Exhibit Shuusou Gyoku (Shuusou Gyoku '99) displayed (23rd Hatoyama Festival)
1999/12/08 Touhou Fantasy Space "Hakurei Shrine" (the old name of the official website) is opened
2000/08/12 Trial Shuusou Gyoku trial released (Comiket 58)
2000/12/30 Game Shuusou Gyoku final version released (Comiket 59)
2001/05/04 Game Torte Le Magic Ver. 1.00 released
2001/08/08 Shuusou Gyoku recorded on TechWin DVD
2001/08/10 Trial Kioh Gyoku trial released (Comiket 60)
2001/12/30 Game Kioh Gyoku release (Comiket 61)

2002[edit]

Date Type Event
2002/03/20 Taito Work begun on Graffiti Kingdom (PS2)
2002/05/-- A CD release of the first five Touhou games added to distribution

2002/07/29 November issue of DOS/V POWER REPORT ships with Embodiment of Scarlet Devil trial version 0.08+
2002/08/11 Game Embodiment of Scarlet Devil released (Comiket 62)
2002/08/11 Music Dolls in Pseudo Paradise (C62 version) released (Comiket 62)
2002/09/05 Watanabe Workshop(now known as French Bread) introduces Touhou in the banner of the website

2002/12/30 Event Comiket 63 (Day 3: one Touhou derivative circle in attendance)
2002/12/30 Trial Perfect Cherry Blossom trial released (Comiket 63)
2002/12/30 Music Dolls in Pseudo Paradise (Popular Version) released (Comiket 63)
