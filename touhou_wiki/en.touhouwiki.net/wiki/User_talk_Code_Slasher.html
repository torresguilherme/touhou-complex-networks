About borrowing lyrics from external sources[edit]

Hi. I talked to the staff on IRC recently. There should be no problem with copying lyrics from other websites or writing them down from the songs. We might as well say that having publicly available lyrics will add to an album's value.

I've also converted the lyrics you've brought into a standard format that uses the template. I used a scripting language to help me with some of the work. The place the lyrics were copied from is now included in the notes. --Sbluen (talk) 17:41, 20 September 2012 (EDT)

Excellent work. The only thing is that some of the pages with lyrics for the songs from TOHO EUROBEAT VOL.2 used the wrong picture. As such, I corrected those pages. Code Slasher (talk) 11:49, 22 September 2012 (EDT)
I modified the script to do nearly everything. It still can't edit wiki pages for you, and it can't do tasks that require differentiation between romanji and English (the eng_only parameter and translated versions of the title). See http://nopaste.info/8314041207.html and if you want to try it out, let me know how it works. --Sbluen (talk) 14:06, 24 September 2012 (EDT)
I've modified the script some more while dealing with special cases, and it is now going to be part of my subpage at User:Sbluen/autoformat. I have used it in the creation of a lot of the songs from the above website. To write scripts for the rest of the lyrics would probably only increase the total amount of effort needed. By the way, you can't just copy the code: you have to copy it from editing window to preserve the whitespace. --Sbluen (talk) 16:45, 27 September 2012 (EDT)
As far as I could tell, I copied the correction from the editing window of one page to the editing window of another? Code Slasher (talk) 16:52, 27 September 2012 (EDT)
I'm sorry I forgot to respond and check your talk page. "By the way, you can't just copy the code: you have to copy it from editing window to preserve the whitespace" is now an outdated statement due to the syntaxhighlight tags. In case you want to know what I'm talking about, it's the fact that the Python programming language requires whitespace to be present and wouldn't have functioned if you copied it improperly. --Sbluen (talk) 16:47, 15 January 2013 (EST)
