Categorizing Untranslated articles[edit]

I've added a new category for lyrics: Category: Untranslated/Lyrics --Tsukihime 15:12, 13 January 2011 (UTC)

Following subcategories to be used if people agree:

  • Category:Untranslated/Lyrics - for music lyrics
  • Category:Untranslated/Dialogue - for any dialogue that occur in games
  • Category:Untranslated/Documentation - for things like game manuals, instructions, game character profiles, etc.

Documentation can be broken down into categories if it is too broad. --Tsukihime 14:43, 14 January 2011 (UTC)
