東方武闘外伝 (Touhou Battle Gaiden) is a clone of Bullet Proof Software's Tetris Battle Gaiden, created by 夏季限定 (Summer Limit). It stars Reimu Hakurei, Marisa Kirisame, Sakuya Izayoi, Youmu Konpaku, Reisen Udongein Inaba, Aya Shameimaru and Cirno, and various other characters make appearances throughout the game.

Gameplay[edit]

Main article: Gameplay

The game is basic VS Tetris with a twist: each character has their own special abilities, which can only be activated through the use of orbs. Orbs can only be obtained by clearing a line which contains them, which appear in certain pieces. Complicating matters is the communal pool of pieces drawn on by both players, only some of which contain orbs.
