Yes, that's what I used to create the pages from yesterday. Would you mind letting me know what unusual things you noticed? Here are the issues I noticed while proofreading them today (or, in some cases, looking at the page history):
    1. Missing credits. I don't know how to handle this, so credits to staff members other than arrangers, vocalists and lyricists have to be added manually - at least for now. Similarly, the script does not preserve information about what circle an artist is from.
    2. One of the Yuuhei Satellite albums had "null" where it should say "砕月". This was due to a problem with the script, but it has been fixed. (I had simply neglected IaMP's pre-battle themes.)
    3. The link to the lyric page of 居るは居ないこの世界＜はこ＞。 on 0と1の円環論理 was originally removed because using the wrong brackets (<>) broke it.
    4. A couple of mistakes from thwiki.cc were corrected.
    5. Using {{lang|ja|<name>}} instead of {{<name>}}. This is not a problem with the script - in fact, it doesn't use {{lang|ja|<name>}} for any artist name. This was just a case of me adding lang|ja| when it wasn't necessary.
I agree that double-checking the info is important, and I appreciate that you're willing to help me with it. Feedback on the Digital Distribution addition[edit]

Hi! Nice seeing you again, and thanks for the feedback on the Digital Distribution addition. I'm not aware of any album that appears on iTunes without also appearing on Google Play Music (or vice versa), but it's probably possible. I've changed how Template:MusicArticle/sandbox handles links to digital distributions - now it only has one parameter, "digital", which takes one string of pre-formatted links. Is that fine, or is there a better way to do it? I could've added parameters for the other sites instead, but I belive that would've made the code pretty ugly... There's also a separate "tdmd" parameter, so we can link to iTunes and Google Play Music without the template assuming that the album has been published via TDMD. I think that's worth keeping either way. -Maths ~Angelic Version~ (talk) 14:17, 9 April 2019 (UTC)

I think that way works, the website parameter work more or less the same so it seems like a natural implementation. I also like the idea of the TDMD having a parameter so this seems good to me, thanks for making the change. Ycdtosa (talk) 01:49, 13 April 2019 (UTC)
Glad you like it. I'll make the same changes to the MusicArticle template. -Maths ~Angelic Version~ (talk) 15:17, 17 April 2019 (UTC)
