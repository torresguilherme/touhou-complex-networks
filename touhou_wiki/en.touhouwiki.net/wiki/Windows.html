Windows

From Touhou Wiki
Jump to navigation Jump to search

Microsoft Windows (Windows for short) is an operating system produced by Microsoft, available in several versions. It's used on personal computers and is one of the most widely-used operating systems in the world. "Windows" is also a term to describe the Touhou Project in general when excluding the PC-98 games.

There have been many versions of Windows, such as Windows 98 (not to be confused with the PC-98) and Windows XP. Windows 10 is the latest edition.

Embodiment of Scarlet Devil (Touhou 6) was the first Touhou Project game developed for this platform, able to play on Windows 98/SE, ME, 2000 and XP. Shuusou Gyoku was technically the first game for this platform, but it's part of the Seihou Project and wasn't directly developed by ZUN. All the games from Embodiment of Scarlet Devil on were made entirely for this platform. They are distinguished from "the PC-98 games" (Touhou 1 to 5) by being published by Team Shanghai Alice, it having been branched off from Amusement Makers.

Overview on the Touhouverse[edit]

Terminology[edit]

"Windows" or "the Windows games" as a term refers to anything from Embodiment of Scarlet Devil onwards in the Touhou Project by the Western community; this includes all games, music CDs and comics. This can also be called "recent Touhou works", "Today's games" or simply "The Touhou Project". The terminology is supposed to distinguish them from the PC-98 era that holds the first five games of the Touhou series. It's also used to reflect the modern music, graphics, complex stories and characters that occur in all the material, which is by many fans considered new or modern.

Setting[edit]

Main article: Canon: Canonicity and comparison between Windows and PC-98

Unlike the PC-98 era of the Touhou Project, many details are available regarding the setting of the Windows games. The project is set in Gensokyo and has many characters, ranging from humans to youkai.

The Windows games are often split into three generations, with Mountain of Faith being the start of the second generation and Double Dealing Character being the start of the third. This is because Mountain of Faith introduced a new game engine that has been used in every Touhou Project shooting game since. Additionally, the plots of the Touhou Project games starting with Mountain of Faith have become more connected to each other, up until Hopeless Masquerade. A unique plot was introduced in Double Dealing Character, making it the third generation for Windows.

Unlike incidents that have happened in the PC-98 games, the incidents that occur are more serious and somewhat complex, often involving a major change in the climate or some suspicious occurrence in Gensokyo. It's up to Reimu or whoever volunteers to go out and solve the issue.

Additional Information[edit]

  • This platform was a fresh start for ZUN as he was no longer limited to 16-color graphics.
  • The last main series game to officially work on Windows 98 is Phantasmagoria of Flower View. Shoot the Bullet and Mountain of Faith will run as well, but show some oddities like missing BGM. From Subterranean Animism on, the games depend on a newer DirectX 9.0c build not supported on Windows 98.
  • The first game built with Windows Vista in mind was Undefined Fantastic Object.
  • The first game built with Windows 7 in mind was Double Spoiler.
  • The first game built with Windows 8 in mind was Double Dealing Character, which also doubled the resolution from 640×480 to 1280×960.
  • The first game built with Windows 10 in mind was Legacy of Lunatic Kingdom.
  • The first game to officially support Windows 10 at the time of release was Hidden Star in Four Seasons.
  • All the games from Embodiment of Scarlet Devil to Phantasmagoria of Flower View are able to play on Windows Vista/7/8, but in some cases, the title screen won't display properly and won't show the image of Reimu Hakurei. For Windows Vista/7, enabling Windows Aero may help.
    • Embodiment of Scarlet Devil will sometimes stop working when launched. This sometimes only happens without Aero.
    • There is a Fullscreen issue for these games when running them on Windows 10. A third-party DLL converter is available that can fix this. See DX8 to DX9 converter.
  • If using Vista or Windows 7, all the games from Double Spoiler onward will disable Aero and switch the theme to "Windows Vista Basic" or "Windows 7 Basic". Windows will switch back to Aero after exiting the game. There exists a patch for Ten Desires to prevent this and allow the game to play in Aero.

Fandom[edit]

  • Windows characters are seen as separate from the PC-98 characters by many fans, excluding the few that returned.
  • An unofficial tool exists to convert the .wav music of all games with ZUN's involvement to .ogg and patch the games to read from those files in order to save disk space.
