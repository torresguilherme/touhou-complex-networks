Nicovideo Encyclopedia

From Touhou Wiki
Jump to navigation Jump to search

Nicovideo Encyclopedia (ニコニコ大百科, Nico nico daihyakka, official English name: Nico Nico Pedia or nicopedia) is a wiki style Website managed by Brazil (未来検索ブラジル, Mirai kensaku burajiru). Its editing stance is somewhere in the middle of Wikipedia and Uncyclopedia. It's one of the major sites that contains vast information about the Touhou Project.

Nico/Pixiv Sorting Project[edit]

Note: once this project is completed, this section will be removed.

These pages needs reviewing and suggested to be merged to their character pages, or otherwise to be deleted. They're outdated, so please review these pages and use this scheme to work out what's factual information and what is not (Use <font style="background-color:green/yellow/orange/red">information</font>):

  • 　 : Info highlighted in this is definite factual info and can go onto the pages. This will also be used when the info is already on the character pages. There maybe some questionable in this, but not alot.
  • 　 : Info highlighted in this is questionable and it will make it hard to tell whether is can go onto the pages or not. It may need to be discussed, but references could be used to make it less questionable.
  • 　 : Info highlighted in this is most likely a definite no, and cannot go onto the pages as there would be no references to prove that the information is correct. It may need to be discussed if necessary as it shouldn't be used straight away.
  • Use strike tag (<s></s>) to indicate the sentence to be "finished for merging", "information already in the page" and "reviewed and determined as unnecessary info".
