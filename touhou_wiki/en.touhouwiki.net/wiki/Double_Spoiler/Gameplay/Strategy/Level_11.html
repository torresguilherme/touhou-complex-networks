Spell Card 11 - 1

Owner

Level 11 — 1
Editor's Comment
Stay on the bottom of the pulsating circle in the background: the bullets travel in a more or less straight line there, so you can pick a stationary safe spot for each salvo, and charge the camera.

Spell Card 11 - 7
儚道「御神渡りクロス」
Fleeting Path "Omiwatari Cross"
Owner
Kanako Yasaka
Level 11 — 7
Editor's Comment
This card can be defeated with sheer luck, but there is a strategy.

The fundamental idea is to capture all four set of bullets in a single shoot, then rinse and repeat.

  1. Lead the 1st set of bullets to the right side of the screen.
  2. Move to the middle-left of the screen, leading the 2nd set of bullets to the middle of the screen.
  3. Dodge to the right a little when the 3rd set arrives. The 1st set of bullets should be splitting.
  4. When the 4th set begins to appear, the 1st set should be arriving. Shoot everything.
There is enough spare time to charge. Also the shoot should clear enough of the 3rd set of bullets so that you can return to the right with ease.
