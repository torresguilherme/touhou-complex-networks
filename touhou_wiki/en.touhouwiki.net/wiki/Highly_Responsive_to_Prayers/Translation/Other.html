English Translation[edit]


Touhou Project Part 1
    Touhou Rei'iden

0. First of all
　　Thank you very much for purchasing the Shrine Maiden action shooting game Touhou
Rei'iden.  Please read through this manual ve~ry carefully
and enjoy the strange, strange world this shrine maiden resides in.

1. System Requirements
 *pc98 / EPSON PC with EGC (VX or later)
　*386 or higher (486(33MHz) or better recommended)
　*550k or more free main memory
　*FM sound card supported (86 is sufficient)
　*Runs off the hard disk

2. Installation
　Insert the floppy disk in the drive, make that drive the current drive, then
enter the following in the command line:

      INSTALL (destination drive letter):

  Example) INSTALL A:   (install to the A drive)

　Press enter and installation will begin. Please make sure that the destination
drive has the necessary 3MB free.

　For manual installations:
　Copy REII.EXE to the destination drive and run it.  REII.EXE is not necessary
to run the game so it may be deleted once installation has completed.

3. Starting the Game
　Change to the directory the game was installed to and input:
 　　GAME
in the command line.
　The title screen will appear and display 4 options:
　　START ...　Start the game.
　　CONTINUE ...　Continue.
  OPTION ... Change all sorts of settings.
　　QUIT ... Return to DOS.

4. Rules of the game

　There are 20 stages, and every 5 stages, a boss will show up.

　*Regular stage
  Flip all the panels within the screen 
  using the Yin-Yang Orb. Once all the panels
  have been flipped, the stage will be cleared. 

　*Boss stage (Stage 5, 10, 15 and 20)
  Damage can only be dealt by hitting the boss
  with the Yin-yang Orb. (When you have 0 lives
  left, the spirit attack can also deal damage)
  Bosses have various kinds of attacks but you  can
  repel white bullets with your exorcism, Shoutenshuu,
  or the start of an ofuda burst. Also, if you hit
  a bullet directly with an ofuda you can destroy it.

  After the 6th stage, there will be 2 routes.

5. Controls

 *Players have their choice for movement controls.
　　　　Arrow Keys　　　　　　　　　Numpad
　　　　　　< >　　　　　　　　　　　　4　　6

　　Player movement is restricted to left and right. 

　*Z key   Throw an ofuda.
   If you hit the Yin-yang Orb it will be deflected.
   Enemy bullets (only white ones) can be destroyed with this attack.

　*X key   Perform an exorcism.
   Both the Yin-yang Orb and enemy bullets can be deflected.
   The faster the Yin-yang Orb comes in the further it'll fly.
　*Left+Right+Z Key
   or
   X+Z　2 times   Uses a spirit attack. 
   You can only bomb as many times as indicated
   by the spirit attack counter in the upper
   part of the screen.
   You will gain a bomb when you collect a bomb
   item or die. 

 *Left or right key + X key
   Slide in the direction pressed.
   You can deflect the Yin-yang Orb with a slide.
   During a slide your hitbox will shrink.

　**Spirit combo attack
   If you do the following during a slide...
   Neutral (not pressing left or right)
      X key   ... Zenshinshoutenshuu (lit. Driving Send-to-heaven Kick)
   Same direction key as the slide +
      X key   ... Jump kick
   Direction key opposite the slide direction +
      X key   ...　Shoutenshuu (lit. Send-to-heaven Kick)

      Z key   ... 8-ofuda burst

   If you time it right, you can press the same buttons again during a Shoutenshuu,
   and chain up to 3 attacks in a row.
   (You can't do this for the Jump Kick or ofuda burst)

　*ESC key　　　Pause the game.
   (While paused, you can return to the title screen.)

6. About items
 P           ... Increases your score.
 Bomb Extend ... Increases your bomb count by 1.


7. Others
   To know about the story, or about detailed controls,
   please read　REIIDEN.TXT after installing.

   If problems such as being unable to
   install the game or unable to run the  
   game due to a broken disc, please contact 
   us at the location written below.

-------------------------------

355 Saitama, Higashi-Matsuyama City,　Motojuku　1-6-5　Nakamura Building 208
      Oota Jun'ya (ZUN)

   And so, the eastern sky burns in violet...
                  in swarms

Japanese[edit]

<toggledisplay showtext="Show original text" hidetext="Hide original text"> 

     　　　東方Ｐｒｏｊｅｃｔ第１弾    　　　　
                  東方靈異伝                   

■０．はじめに
　　この度は、
　　　　巫女さんアクションシューティング
　　　　　　　　　東方靈異伝
　　をお買上頂きまことにありがとうございます。
　このマニュアルをよ～く読んで、不思議な不思議な
　巫女さん世界を楽しんで下さいね。

■１．動作環境
　●pc98 / EPSON PC でＥＧＣ搭載機種（ＶＸ以降）
　●要３８６以上（４８６（３３ＭＨｚ）以上推奨）
　●空きメインメモリ５５０ｋ以上
　●ＦＭ音源（８６相当）対応
　●ハードディスク専用

■２．インストール方法
　フロッピーディスクをドライブにいれ、そのドライ
　ブをカレントドライブにして、コマンドラインから

      INSTALL インストール先のドライブ名:

  例）INSTALL A:   （Ａドライブにインストール）

　と入力しリターンキーを押して実行して下さい。
　インストール先ドライブの空き容量が、作業中も
　含めて、約３メガ必要ですので、注意して下さい。

　（手動でインストールする場合）
　REII.EXE を、インストール先へコピーし実行して
　下さい。REII.EXE は不要ですので、削除していい
　です。

■３．起動方法
　インストール先のディレクトリをカレントにして、
　コマンドラインより、
 　　GAME
  と入力して下さい。
　タイトル画面が表示され選択肢が４項目表示され
　ます。
　　ＳＴＡＲＴ　　　・・・　ゲームを始めます。
　　ＣＯＮＴＩＮＵＥ・・・　コンティニューです
　　ＯＰＴＩＯＮ　　・・・　各種設定を出来ます。
　　ＱＵＩＴ　　　　・・・　Ｄｏｓに戻ります。

■４．このゲームのルール

　全２０面で、各５面おきにボスが登場します。

　●通常面
　　画面内のパネルを、陰陽玉を通過させる事によっ
　　てめくれます。全てのパネルをめくることによっ
　　て、クリアとなります。

　●ボス面（５、１０、１５、２０面）
　　陰陽玉を当てる事しかダメージをあたえられませ
　　ん。（残り人数０人の時のみ靈撃でもダメージを
　　与えられます）
　　　ボスは色々な攻撃を仕掛けてきますが、白い弾
　　は、お払いや昇天蹴、お札乱れうちの出始めで、
　　跳ね返すことができ、また、お札を直接当てるこ
　　とで打ち消すことが、出来ます。

　　６面以降は、ルートが２つあります。


■５．操作方法

　●それぞれ自機移動です。

　　　　カーソル　　　　　　　　　テンキー
　　　　　←→　　　　　　　　　　４　　６

　　自機移動は、左右しかありません。

　●Ｚキー　　　お札を投げます。
　　　　陰陽玉に当てると跳ね返すことが出来ます。
　　　　敵弾（白い弾のみ）を消すことが出来ます。

　●Ｘキー　　　お払いをします。
　　　　陰陽玉、敵弾、共に跳ね返します。
　　　　陰陽玉は、引き付けたほうが高く飛びます。

　●左右両方押し＋Ｚキー
　　　　もしくは
　　　Ｘ＋Ｚ　２回　　　霊撃を使います。
　　　　画面上の、霊撃の所の回数分だけ使えます。
　　　　アイテムを取るか、自分がやられると、
　　　　一個増えます。

　●左右キーどちらか＋Ｘキー
　　　　押した方向にスライディングをします。
　　　　陰陽玉のみ、跳ね返せます。
　　　　この時、あたり判定は、小さくなります。

　☆霊連撃について
　　スライディング中に次のボタンを押すと・・・
　　　ニュートラル（左右どちらも押さない）
　　　　　Ｘキー　　・・・　前進昇天蹴
　　　進行方向キー
　　　　　Ｘキー　　・・・　跳び蹴り
　　　反対方向キー
　　　　　Ｘキー　　・・・　昇天蹴

　　　　　Ｚキー　　・・・　お札８連乱れ撃ち

　　昇天蹴中さらにタイミングよくボタンを押せば、
　　３回まで連続で技が出せます。
　（跳び蹴り、乱れ撃ちをすると続きが出ません。）

　●ＥＳＣキー　　　ポーズをします。
　　（ポーズ中から、タイトルにもどれます。）

■６．アイテムについて
　Ｐ　　　　 　・・・　得点です。
　Bomb Extend　・・・　ボムが一個増えます。


■７．そのほか
　　ストーリーや、細かな操作方法などは、
  インストール後　靈異伝.TXTを、お読みください。

　　万一、インストール出来ない、とかディスク不良
　とかが原因でゲームが出来ない、といった困った事
　が在りましたら、お手数ですが下記の所へご連絡を
　してください。

－－－－－－－－－－－－－－－－－－－－－－－－
３５５
埼玉県  東松山市　元宿　1-6-5　中村ビル 208号室
　　　　　　　　太田順也（ＺＵＮ）

　　そして、東方の空はむらさきにもえて・・・
　　　　　　　　　　　　　　　　　　うじゃうじゃ

</toggledisplay>
